### Eratosthenes 素数筛选

```c++
for (int i = 2; i <= n; i++) {
    is_prime[i] = true;
}
for (int i = 2; i * i <= n; i++) {
    if (is_prime[i]) {
        for (int j = i * i; j <= n; j +=i) {
             is_prime[j] = false;
        }
    }
}
```

优化后时间复杂度比O(NloglogN)还要低
